# STD стандартна бібліотека шаблонів  

## Завдання 

- ознайомитися із поточним проєктом (консольний застосунок)
- документація STD - https://en.cppreference.com/w/cpp/header (тут наведені все класи та всі оператори та методи + приклади)
- створити графічний інтерфейс для використання шаблонів STD - двостороння черга, список, вектор, черга з пріорітетом, звичайна черга та стек. Мінімальні вимоги:  
    - користувач повинен мати змогу обрати шаблон (із переліку)
    - додати елементи до шаблону
    - передивлятися додані елементи
    - видаляти елементи 
- оформити звіт та надіслати викладачу
